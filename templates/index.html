Construction Cost Estimation and Project Analytics

Welcome to the Construction Cost Estimation and Project Analytics page! Here, you can analyze various project parameters to assess potential costs for construction projects.

Project Parameters for Cost Estimation

Understanding the relationship between project data and costs can provide valuable insights for effective project management. Please refer to the following key project indicators:

  • Commodity Code: The unique code representing the commodity.
  • Item Description: Description of the item.
  • Qty: Quantity of the item.
  • PE Amount: Pre-Engineering amount for the project.
  • BM Amount: Bill of Materials amount for the project.
  • LB hrs: Labor hours required.
  • LB Amount: Labor cost for the project.
  • CE Amount: Cost Estimate amount for the project.
  • Major SC Amount: Major Subcontractor amount for the project.
  • Fuel usage (L): Fuel usage in liters.
  • Attribute 1: Additional attribute for the project.
  • Attribute 2: Additional attribute for the project.
  • Attribute 3: Additional attribute for the project.
  • Project Number: Unique project number.
  • Total New: New total cost for the project.
  • Total: Overall total cost for the project.
  • Single Unit Price: Price per single unit.
  • Epic Embodied Carbon: Epic Embodied Carbon in kgCO2e.
  • AUS LCI Embodied Carbon: AUS LCI Embodied Carbon in kgCO2e.
  • Carbon Allowance: Allocated carbon budget.
  • Construction Carbon: Carbon emitted during construction.
  • Default PE Unit Price: Default unit price for Pre-Engineering.
  • Default BM Unit Price: Default unit price for Bill of Materials.
  • Default LB Unit Hrs: Default labor unit hours.
  • Default SC Unit Rate: Default subcontractor unit rate.
  • Project Name: The name of the project.
  • Greenfield/ Brownfield: Greenfield or Brownfield classification of the project.
  • Client: Client commissioning the project.
  • Market Sector/Industry: The sector or industry of the project.
  • Lat: Latitude of the project location.
  • Long: Longitude of the project location.
  • Delivery Method: Method of delivery for the project.
  • Item Type: Type of the item in the project.
  • Flag: Special flag for the project.
  • Coordinates: Geographical coordinates of the project.
  • State: State where the project is located.
  • City: City where the project is located.
  • Suburb: Suburb where the project is located.

For analysis, please enter the above parameters as comma-separated values.
